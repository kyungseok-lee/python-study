# 🌱 초급 연습 문제

> Python 의 기초를 익히는 연습 문제입니다.

## 📚 문제 목록

### 01. 환경 설정과 기초 ⭐

#### 문제 1.1 — Hello, 당신의 이름!
사용자에게 이름을 입력받아 인사하는 프로그램을 작성하세요.
```
이름을 입력하세요: 홍길동
안녕하세요, 홍길동님!
```

#### 문제 1.2 — 자기소개
이름, 나이, 취미를 입력받아 자기소개를 출력하세요.
```
=== 자기소개 ===
이름: 홍길동
나이: 25
취미: 코딩
```

### 02. 변수와 타입 ⭐

#### 문제 2.1 — BMI 계산기
키(m)와 몸무게(kg)를 입력받아 BMI 를 계산하세요. (소수점 2자리)
```
체중(kg): 70
키(m): 1.75
당신의 BMI는 22.86 입니다.
```

#### 문제 2.2 — 온도 변환
섭씨를 입력받아 화씨와 켈빈으로 변환하세요.
- 화씨: `°C × 9/5 + 32`
- 켈빈: `°C + 273.15`

### 03. 연산자 ⭐⭐

#### 문제 3.1 — 시간 변환
초 단위 입력을 시:분:초 로 변환하세요.
```
초: 3725
1시간 2분 5초
```

#### 문제 3.2 — 거스름돈
받은 돈과 가격을 입력받아 거스름돈을 동전 단위(500, 100, 50, 10원)로 분리하세요.

#### 문제 3.3 — 윤년 판별
연도를 입력받아 윤년인지 판별하세요.
- 4의 배수: 윤년
- 100의 배수: 윤년 아님
- 400의 배수: 윤년

### 04. 제어문 ⭐⭐⭐

#### 문제 4.1 — FizzBuzz ⭐⭐
1-100 중:
- 3의 배수: "Fizz"
- 5의 배수: "Buzz"
- 15의 배수: "FizzBuzz"
- 그 외: 숫자

#### 문제 4.2 — 구구단
2단부터 9단까지 출력하세요.
```
2 x 1 = 2
2 x 2 = 4
...
```

#### 문제 4.3 — 별 찍기
정수 n을 입력받아 다음과 같이 출력하세요.
```
n=5:
    *
   ***
  *****
 *******
*********
```

#### 문제 4.4 — 소수 찾기 ⭐⭐⭐
2부터 100까지의 모든 소수를 출력하세요.

#### 문제 4.5 — 약수 구하기
숫자를 입력받아 그 숫자의 모든 약수를 출력하세요.

### 05. 함수 ⭐⭐⭐

#### 문제 5.1 — 두 수 사이의 합
정수 a, b 를 받아 a 부터 b 까지의 합을 반환하는 함수를 작성하세요.

#### 문제 5.2 — 팩토리얼
n 의 팩토리얼(n!)을 계산하는 함수를 작성하세요.

#### 문제 5.3 — 회문 판별
문자열이 회문인지 판별하는 함수를 작성하세요.
- `is_palindrome("level")` → True
- `is_palindrome("hello")` → False

#### 문제 5.4 — 가변 인자 평균
여러 숫자를 받아 평균을 계산하는 함수를 작성하세요.
```python
average(1, 2, 3, 4, 5)  # → 3.0
average(10)              # → 10.0
average()                # → 0 또는 None
```

#### 문제 5.5 — 최대값
가변 인자로 받은 숫자 중 최대값을 반환하세요. (built-in max 사용 X)

### 06. 자료구조 ⭐⭐⭐

#### 문제 6.1 — 평균 점수
점수 리스트를 받아 평균을 계산하세요.

#### 문제 6.2 — 두 리스트 합치기
두 리스트를 받아 중복 없이 정렬된 리스트를 반환하세요.
```python
merge([1, 3, 5], [2, 3, 4])  # → [1, 2, 3, 4, 5]
```

#### 문제 6.3 — 단어 빈도
문장을 받아 각 단어의 등장 횟수를 딕셔너리로 반환하세요.
```python
count_words("the quick brown fox the lazy dog")
# {'the': 2, 'quick': 1, 'brown': 1, ...}
```

#### 문제 6.4 — 전화번호부
이름과 전화번호를 저장/검색/삭제하는 간단한 전화번호부를 만드세요.

#### 문제 6.5 — 가장 빈도 높은 요소
리스트에서 가장 자주 등장하는 요소를 찾으세요.

### 07. 문자열 ⭐⭐

#### 문제 7.1 — 단어 뒤집기
"Hello World" → "World Hello"

#### 문제 7.2 — 모음 카운트
문자열에서 모음(a, e, i, o, u)의 개수를 세세요.

#### 문제 7.3 — 압축
"aaabbbccd" → "a3b3c2d1"

#### 문제 7.4 — 비밀번호 강도
비밀번호가 다음 조건을 만족하는지 체크:
- 8자 이상
- 대문자 포함
- 숫자 포함

#### 문제 7.5 — 문장 첫 글자 대문자
"hello world. python is fun." → "Hello world. Python is fun."

### 08. 입출력 ⭐⭐⭐

#### 문제 8.1 — 입력 합계
공백으로 구분된 숫자들을 입력받아 합을 출력하세요.

#### 문제 8.2 — 파일 줄 수
파일 경로를 입력받아 그 파일의 줄 수를 출력하세요.

#### 문제 8.3 — 파일 복사
원본 파일과 대상 파일 경로를 받아 복사하세요.

#### 문제 8.4 — 단어 검색 (grep)
파일 경로와 단어를 받아 해당 단어가 포함된 줄을 모두 출력하세요.

#### 문제 8.5 — 통계 출력
숫자가 한 줄씩 들어있는 파일을 읽어 합/평균/최소/최대를 출력하세요.
