# 🌳 고급 연습 문제

> 깊이 있는 Python 기능을 다루는 문제입니다.

## 📚 문제 목록

### 01. 데코레이터 ⭐⭐⭐⭐

#### 문제 1.1 — @timer
실행 시간 측정 데코레이터.

#### 문제 1.2 — @log
함수 호출 시 인자와 반환값을 로깅.

#### 문제 1.3 — @cache (직접 구현)
lru_cache 없이 결과 캐싱 데코레이터.

#### 문제 1.4 — @validate
함수 인자가 모두 정수인지 검증.

#### 문제 1.5 — @retry
실패 시 최대 N번 재시도. 지연 시간 옵션.

#### 문제 1.6 — 다중 데코레이터
@timer 와 @cache 를 함께 적용해 fibonacci 최적화.

### 02. 메타클래스 ⭐⭐⭐⭐⭐

#### 문제 2.1 — 자동 등록
하위 클래스를 dict 에 자동 등록 (`__init_subclass__`).

#### 문제 2.2 — PositiveNumber 디스크립터
양수만 허용하는 디스크립터.

#### 문제 2.3 — WriteOnce
값을 한 번만 설정할 수 있는 디스크립터.

#### 문제 2.4 — __slots__ 메모리
일반 클래스 vs __slots__ 백만 개 객체 메모리 비교.

#### 문제 2.5 — 메타클래스 싱글톤
메타클래스로 싱글톤 구현.

### 03. 동시성 ⭐⭐⭐⭐

#### 문제 3.1 — 병렬 카운터
스레드 5개 각 1000번 카운터 증가. Lock 사용 전후 비교.

#### 문제 3.2 — URL 상태 확인
URL 리스트 HTTP 상태를 ThreadPoolExecutor 로 병렬 확인.

#### 문제 3.3 — 소수 카운트 (multiprocessing)
1-10,000,000 소수 개수, 4 프로세스 병렬 계산.

#### 문제 3.4 — 생산자-소비자
3 생산자 / 2 소비자, Queue 로 통신.

#### 문제 3.5 — 데드락 시뮬레이션
2 lock 으로 데드락 만들고 해결.

#### 문제 4.5 — 타임아웃
