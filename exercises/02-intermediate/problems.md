# 🌿 중급 연습 문제

> Python 의 핵심 기능을 활용하는 문제입니다.

## 📚 문제 목록

### 01. 함수 심화 ⭐⭐⭐

#### 문제 1.1 — 람다와 정렬
딕셔너리 리스트를 특정 키로 정렬하는 함수를 작성하세요.
```python
sort_by(data, "age")           # 나이순 오름차순
sort_by(data, "age", reverse=True)
```

#### 문제 1.2 — 함수 합성
두 함수를 합성하는 `compose(f, g)` 를 구현하세요. `compose(f, g)(x) == f(g(x))`

#### 문제 1.3 — 클로저 카운터
`make_counter(start=0, step=1)` 을 호출하면 호출할 때마다 증가하는 카운터를 만드세요.

#### 문제 1.4 — 메모이제이션 데코레이터
`@memoize` 데코레이터를 직접 구현하세요. (lru_cache 사용 금지)

#### 문제 1.5 — map/filter/reduce
`[1, 2, 3, 4, 5]` 에서 짝수의 제곱의 합을 map/filter/reduce 만으로 구하세요.

### 02. OOP ⭐⭐⭐⭐

#### 문제 2.1 — BankAccount
입금, 출금, 잔액 확인이 가능한 BankAccount 클래스를 만드세요. 잔액 부족 시 InsufficientFundsError 발생.

#### 문제 2.2 — 도형 다형성
Shape 부모 클래스를 만들고 Rectangle, Circle, Triangle 자식을 만들어 면적/둘레를 계산하세요.

#### 문제 2.3 — Vector 클래스
2D 벡터: 덧셈, 뺄셈, 스칼라 곱, 내적, 크기, 정규화 구현.

#### 문제 2.4 — 도서관 시스템
Book, Library 클래스. 책 추가, 대출, 반납. 같은 책은 여러 권 가능.

#### 문제 2.5 — dataclass Product
@dataclass 로 Product 클래스. 가격, 카테고리별 정렬/필터링.

### 03. 모듈 ⭐⭐⭐

#### 문제 3.1 — 계산기 모듈
사칙연산 함수가 있는 `calculator.py` 모듈을 만들고 사용하세요.

#### 문제 3.2 — shapes 패키지
`shapes/circle.py`, `shapes/rectangle.py` 를 만들고 `from shapes import Circle, Rectangle` 가능하게.

#### 문제 3.3 — 가상 환경
가상 환경에서 requests + beautifulsoup4 설치 후 requirements.txt 생성.

#### 문제 3.4 — 날짜 차이
datetime 으로 두 날짜 사이의 일수, 시간 차이 계산.

#### 문제 3.5 — 단어 빈도 (Counter)
파일 단어 빈도 top 10 출력.

### 04. 예외 처리 ⭐⭐⭐
