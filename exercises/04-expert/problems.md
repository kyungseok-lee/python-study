## 🚀 캡스톤 프로젝트 (택 1)

각 프로젝트는 다음을 포함해야 합니다:
- ✅ 완전한 README
- ✅ pyproject.toml
- ✅ 80%+ 테스트 커버리지
- ✅ 타입 힌트
- ✅ CI/CD
- ✅ 의존성 주입
- ✅ 에러 처리 및 로깅
- ✅ Docker 화 (선택)

### 프로젝트 A: 블로그 플랫폼 🌐
**기술**: FastAPI + SQLAlchemy + Alembic + JWT

**기능**:
- 사용자 인증 (회원가입/로그인)
- 게시물 CRUD
- 댓글 시스템
- 태그/카테고리
- 좋아요 기능
- 검색 (제목/본문)
- 파일 업로드 (이미지)
- 페이지네이션
- 권한 관리 (작성자/관리자)

**고급 기능** (택):
- 실시간 알림 (WebSocket)
- 마크다운 → HTML 변환
- 이메일 알림
- 캐싱 (Redis)

### 프로젝트 B: 데이터 파이프라인 📊
**기술**: pandas + SQLAlchemy + APScheduler

**기능**:
- 다양한 소스 (CSV/API/DB) 데이터 수집
- 변환 (정제, 집계)
- 적재 (DB, 파일)
- 스케줄링 (정기 실행)
- 에러 처리 및 알림
- 로깅 및 모니터링
- 데이터 검증 (great_expectations)

**보너스**:
- Airflow DAG 으로 마이그레이션
- Streamlit 대시보드
- Slack 알림

### 프로젝트 C: 머신러닝 API 🤖
**기술**: FastAPI + scikit-learn + Docker

**기능**:
- 모델 학습 스크립트
- 모델 평가
- API 서빙 (예측)
- 모델 버전 관리
- A/B 테스트
- 모니터링 (정확도, 지연시간)

**보너스**:
- Kubernetes 배포
- MLflow 통합

### 프로젝트 D: CLI 도구 🛠
**기술**: Click + Rich + Typer

선택하세요:
- **dotfile manager**: 점 파일 관리 도구
- **password manager**: 암호화된 비밀번호 관리
- **task tracker**: 할 일 관리 (포모도로 타이머)
- **git 도우미**: 자주 쓰는 git 명령 자동화
- **clipboard manager**: 클립보드 히스토리

**필수**:
- PyPI 배포 가능한 형태
- 컬러풀한 출력 (Rich)
- 설정 파일 (~/.config/)
- 백업/복원 기능

### 프로젝트 E: 실시간 채팅 💬
**기술**: FastAPI + WebSocket + Redis

**기능**:
- 다중 채팅방
- 사용자 인증
- 실시간 메시지
- 메시지 영속화 (DB)
- 온라인 사용자 목록
- 이미지/파일 전송 (선택)

### 프로젝트 F: 게임 🎮
**기술**: Pygame 또는 텍스트 기반

**아이디어**:
- 텍스트 RPG (스토리, 전투, 인벤토리)
- 던전 크롤러
- 2048 (CLI)
- 카드 게임 (포커, 블랙잭)
- 미니 MMO (소켓 통신)
